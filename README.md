
## Cloning

```bash
$ git clone https://github.com/aceiny/Micro-Hack-Hackthon-.git
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
